# Changelog

## 0.1.0

- Initial Rust starter kit.
- Added Axum server, typed configuration, CORS, tracing, and graceful shutdown.
- Added local Flowless Bridge validation with retries, timeout, secret header, and compatibility body secret support.
- Added auth middleware for required, optional, and admin-only routes.
- Added local TTL cache with optional Redis support.
- Added SQLx database pool and health checks.
- Added trust token key generation, signing, verification, and expiry handling.
- Added public, protected, optional, profile, admin, health, and mock task routes.
- Added Docker, compose, docs, tests, and implementation plan.
